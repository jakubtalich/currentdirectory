# currentdirectory
Get the 📂 current directory, 🪐 hostname and 🌐 IP address

Written in Nim 👑

- **Supported OSes:** Linux 🐧, FreeBSD 😈, Solaris ☀️

- **Building:**
  `$ cd src/ && nim c -d:release currentdirectory.nim`

- **Run options:**
  - `currentdirectory`
  - `currentdirectory --help`
  - `currentdirectory --version`
  - `currentdirectory --license`
